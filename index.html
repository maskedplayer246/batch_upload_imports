Simple JSON card downloads for fans, players, collectors, and non-developers. No coding. No API setup. Just choose a set and click download.

These buttons are designed for GitHub Pages. They download the JSON file directly instead of sending people to a confusing code preview page.

How to Use These Files

  1. Pick the set you want.
  2. Click the Download JSON button.
  3. The file should save to your Downloads folder or ask where to save, depending on your browser settings.
  4. Open the app or tool you want to use.
  5. Upload the JSON file when prompted.

No coding required.
