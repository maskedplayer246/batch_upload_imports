# Pokémon TCG JSON Card Downloads

Welcome! This page is a simple, non-developer-friendly place to find Pokémon TCG card data in **JSON format** for quick use in compatible tools and fan projects.

These files are intended to make it easy for players, collectors, testers, and community members to download card data without needing to use code, APIs, or developer tools.

> Credit for the JSON card data goes to the **Pokémon TCG Developers** and the Pokémon TCG data/API community.

## How to Use These Files

1. Pick the set you want.
2. Click **Download JSON**.
3. Save the file to your computer.
4. Open PokeBowl.
5. Upload the JSON file when prompted.

No coding required.

## About This Page

This page is meant to make Pokémon TCG JSON card data easier to access for fans and non-developers.

The JSON files are intended for quick importing into fan tools, deck tools, simulators, collection tools, testing apps, and other community projects.
