## About This Page

This page is made for fans who want quick access to Pokémon card data in **JSON format**.

The goal is to make the files easy to find and easy to use, especially for people who are not developers.

Credit for the JSON files goes to the **Pokémon TCG Developers** and the Pokémon TCG data/API community.

# How to Use These Files

1. Pick the set you want.
2. Click the **Download JSON** button.
3. Save the file to your computer.
4. Open the app or tool you want to use.
5. Upload the JSON file when prompted.

No coding required.

# Editing the Download Buttons

Every download button currently points to `#`.

To connect a button to a real JSON file, replace the `#` with the file path.

Example:

```markdown
[![JSON][main-json-badge]](files/base-set.json)
```

Example for Pocket:

```markdown
[![JSON][pocket-json-badge]](files/genetic-apex.json)
```

Example for all cards:
